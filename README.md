# Sales-Database-Management-System-
Sales Database Management System

## Overview
This project is a SQL-based Sales Database Management System developed using MySQL. It manages customers, products, employees, orders, and order details while performing sales analysis using SQL queries.

## Features
- Customer Management
- Product Management
- Employee Management
- Order Management
- Sales Analysis
- Revenue Reports

## Technologies Used
- MySQL
- MySQL Workbench
- SQL

## Database Tables
- Customers
- Products
- Employees
- Orders
- Order_Details

## SQL Concepts Used
- DDL & DML
- Constraints
- Joins
- GROUP BY & HAVING
- Aggregate Functions
- Subqueries
- Window Functions
